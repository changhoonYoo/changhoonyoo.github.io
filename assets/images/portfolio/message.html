✉️ 문자 발송 웹서비스

🚩 Description

프로젝트 소개 대량 문자를 발송하는 웹서비스로 SpringBoot RestAPI 백엔드 서버 개발
개발 기간 2023.10 ~ 2024.03
직책 백엔드 팀장
담당 업무
  • 회원가입 - 본인인증 API 연동
  • 결제 - PG API 연동
  • 회원정보 - 회원정보 변경, 비밀번호 변경, 회원 탈퇴
  • 주소록 - 주소록 CRUD
  • 문자 발송 - 즉시 전송, 예약 전송, 반복 전송 기능, 문자 발송 플랫폼 연동
  • 이력 관리 - 결제 내역 관리, 문자 발송 이력 관리
  • 구독 - 구독 서비스 설계, DB 설계, 구독 상태 관리
  • 채팅(웹, 단말기) - 구독 회원 간 채팅 기능, 단말기와의 문자 발송 및 수신 채팅화
  • ARS 서비스 - ARS 대응 시나리오 설정
  • Google TTS/STT 연동
운영 체제 Linux
개발 툴 Intellij, Postman, DBeaver, WinSCP, Mobaxterm, VSCode, JMeter
개발 스킬 Java, JPA, Jwt, Websocket, Spring Security, Docker, SQL, PL/SQL, Apache, Nginx
데이터베이스 PostgreSQL, Oracle, Mysql, Redis

🚩 Images

본인인증 API 연동

☑️ 개인정보(CI/DI)의 평문 노출을 피하는 설계로 개인정보 보호에 있어 안정성있게 구현했습니다.

결제 API 연동

☑️ 결제 성공 시 이력과 함께 회원데이터에 금액을 충전하고, try-catch 구문으로 실패 시 환불 처리되는 로직을 구성했습니다.

Websocket을 활용한 채팅 기능

☑️ Stomp 프로토콜로 수신 문자 메시지에 대한 실시간 알림과 회원간의 채팅을 구현하였습니다.

Linux 환경에서의 운영

☑️ 기본적인 리눅스 명령어를 다룰 줄 알며, 쉘 스크립트를 사용하여 웹서버를 운영하였습니다.

JMeter를 활용한 부하테스트

☑️ JMeter로 API를 안정적으로 고도화 하였습니다.

Apache, Nginx

☑️ 운영 웹서버는 Apache로 운영하였고, stage 웹서버는 Nginx를 운영하였습니다.

대량의 메시지 발송 요청

☑️ 대량 메시지 발송을 비동기 처리하여 사용자 경험을 크게 개선하였습니다.

엑셀 첨부로 주소록 등록

☑️ poi 라이브러리를 활용하여 엑셀 내 내용을 데이터베이스로 적재하는 과정에서 배치 처리를 하여 메모리 사용량을 줄이고 성능을 최적화하였습니다.

Google TTS/STT 연동

☑️ 요청 파라미터를 세분화하여 클라이언트에서 커스텀하기 쉽게 구조를 만들었습니다.

구독 서비스의 설계

☑️ 서비스의 취지를 고려하며 확장성있게 설계하였습니다.
