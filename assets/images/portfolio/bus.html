🚌 버스 예매 프로그램

🚩 Description

프로젝트 소개 Swing 라이브러리와 JDBC를 활용하여 데이터의 흐름을 이해하는 버스 예매 프로젝트
개발 기간 2023.02.07 ~ 2023.02.19 (약 2주)
참여 인원 4명
담당 역할 팀장
담당 업무
  • ID / PASSWD 찾기 UI
  • 마이 페이지, 자주 묻는 질문 UI
  • 좌석 선택 UI 및 기능
  • 예매 내역 UI 및 기능
  • 결제 UI 및 기능
  • 관리자 UI 및 기능
운영 체제 Windows 10
개발 툴 Eclipse, SQL Developer
개발 스킬 Java, Java Swing, JDBC, SQL
데이터베이스 Oracle
