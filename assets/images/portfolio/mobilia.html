🛏️ Mobilia 가구 쇼핑몰

🚩 Description

프로젝트 소개 Spring Framework에서 MVC2 패턴으로 개발한 가구 쇼핑몰 프로젝트
개발 기간 2023.05.08 ~ 2023.06.02 (약 4주)
참여 인원 4명
담당 역할 부팀장
담당 업무
  • Main
    • - header/body/footer UI
    • - 카테고리 별 상품 리스트 표출
    • - 스크롤 버튼
    • - 고정 헤더 및 슬라이드 배너
    • - Sold Out 이벤트
  • Product List
    • - 상품 DB설계
    • - 상품 대분류/소분류별 리스트 및 페이징
    • - 선택 상품 정렬 및 표출
    • - 할인율 계산
  • Product Info
    • - 상품 문의 DB설계
    • - 상품 정보/후기 UI
    • - 상품 메뉴별 스크롤 버튼
    • - 상품 후기 및 페이징
    • - 후기 권한
  • Admin
    • - 관리자 로그인/메인 각 카테고리 UI
    • - 관리자 DB설계
    • - 관리자 상품 관리
    • - 관리자 후기 관리
  • Find ID/PW
    • - 임시 비밀번호 생성 및 이메일 발송
  • MyShop
    • - UI 및 최근 본 상품
운영 체제 Windows 10
개발 툴 Eclipse, SQL Developer
개발 스킬 Java, Springboot, Mybatis, SQL, Javascript, jQuery, HTML, CSS
